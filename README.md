vipm-cli
==========================

A simple CLI.

## Install

```
  $   git clone https://github.com/yyfrontend/vipm-cli.git
  $   cd vipm-cli
  $   npm install
  $   npm link
```

## Usage

### vipm init

``` bash
$ vipm init
```

### vipm dev

``` bash
$ vipm dev(d) [env]
```

### vipm build

``` bash
$ vipm build(b) [env]
```
